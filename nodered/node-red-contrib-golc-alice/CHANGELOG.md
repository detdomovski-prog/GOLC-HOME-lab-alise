# Changelog

## 0.1.36 (2026-03-23)

### 🔧 Fixes — golc-auth-simple

- **Убран `fetch()`** — заменён на встроенные `http`/`https` модули. Теперь нода работает на Node.js >= 14 (раньше крашилась на Node < 18)
- **Авто re-auth при истечении токена** — добавлена проверка времени жизни токена (23 часа). При истечении — автоматическая повторная авторизация
- **Добавлен `node.on('close')`** — корректный cleanup при редеплое/удалении ноды, предотвращает утечки и висящие запросы
- **Промежуточный статус** — синий индикатор «авторизация...» во время запроса
- **`req.destroy()` при таймауте** — корректно закрывает сокет, не оставляя висящих соединений
- **Сброс токена при ошибке** — невалидный токен очищается, следующий input вызовет re-auth
