# 📊 Отчёт о проделанной работе

**Проект:** Интеграция DIY устройств с Алисой через GoSlam Lab
**Цель:** Создание собственного backend + API для управления устройствами и интеграции с Яндекс Алисой

---

## 🧩 1. Начало проекта

На старте стояла задача:

* сделать **свою систему умного дома**
* без готовых платных сервисов
* с возможностью подключать **DIY устройства (ESP / Arduino)**
* с управлением через **Алису**

---

## 🖥 2. Развёртывание инфраструктуры

### 📦 Proxmox

Был установлен **Proxmox** как основа для всей системы:

* создание виртуальных машин и контейнеров
* изоляция сервисов
* удобное управление сервером

👉 В Proxmox развернули:

* Node-RED
* backend/API
* дополнительные сервисы

---

## 🌐 3. Настройка доступа извне

### ☁️ Cloudflare Tunnel

Так как использовался **серый IP**, доступ сделали через Cloudflare:

* установлен `cloudflared`
* выполнено:

  * `cloudflared tunnel login`
  * `cloudflared tunnel create`
* настроен туннель на локальный сервер

👉 Результат:

* доступ к API извне без белого IP
* безопасное подключение
* стабильный endpoint для Яндекса

---

## 🔧 4. Backend и логика (Node-RED)

Основная логика реализована в **Node-RED**:

### Что было сделано:

* созданы HTTP endpoints

* реализованы обработчики:

  * `/user/devices`
  * `/user/devices/query`
  * `/user/devices/action`

* настроена авторизация (token)

---

### 🔗 Пример логики:

* Алиса → запрос в API
* Node-RED:

  * принимает запрос
  * обрабатывает
  * отправляет ответ
* при необходимости → отправляет команду устройству

---

## 🤖 5. Интеграция с Яндекс Алисой

Была реализована работа через **Яндекс Smart Home API**

### Основные этапы:

* регистрация навыка в Яндекс Диалоги
* настройка:

  * webhook URL (через Cloudflare)
  * авторизация

---

### Реализованные методы API:

* `devices` — список устройств
* `query` — получение состояния
* `action` — управление

---

### ⚠️ Проблемы и решения:

* ошибка `Cannot GET /devices/query`
  👉 решено добавлением правильного endpoint в Node-RED

* ошибка `401 Unauthorized`
  👉 исправлена настройкой токена

---

## 🔌 6. Подключение устройств (DIY)

Реализована возможность подключения **самодельных устройств**:

### Подход:

* устройство (ESP8266 / ESP32)
* подключение к Wi-Fi
* взаимодействие с API

---

### Пример:

* устройство отправляет запрос в Node-RED
* получает команду (ON / OFF)
* выполняет действие

---
