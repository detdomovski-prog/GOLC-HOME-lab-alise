# GOLC HOME

Интеграция Яндекс Алисы, Smart Home backend и Node-RED нод для GOLC HOME.

## Статус

**Ведётся бета-тестирование.**

Проект находится в активной разработке.
Структура, API, поведение нод и состав пакетов могут меняться.

## Что входит в проект

- `backend/oauth` — OAuth и backend для интеграции с Яндексом
- `backend/nodered-api` — API для связи backend и Node-RED
- `nodered/node-red-contrib-golc-alice` — Node-RED пакет с нодами
- `routes`, `controllers`, `services` — основная серверная логика

## Для чего этот репозиторий

- тестирование архитектуры интеграции
- разработка и проверка Node-RED нод
- обкатка backend-логики для умного дома
- подготовка к стабильному релизу

## Важно

- текущие версии считаются тестовыми
- обратная совместимость между бета-версиями не гарантируется
- перед production-использованием требуется дополнительная проверка
